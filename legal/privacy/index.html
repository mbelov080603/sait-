Политика конфиденциальности

Эта страница задаёт юридический каркас для обработки обращений с сайта. Часть реквизитов оператора ещё ждёт подтверждения брендом и отмечена как placeholder.

1. Общие положения

Сайт Global Basket использует контактные данные пользователя только для обработки запросов, обратной связи, B2B-обращений, сервисных вопросов и сопутствующих коммуникаций.

До публикации подтверждённых юридических данных оператор персональных данных обозначен на сайте как placeholder. После получения подтверждения этот раздел должен быть заполнен официальными реквизитами без изменения структуры документа.

2. Какие данные могут обрабатываться

Имя, телефон, email, компания, город, тип бизнеса, комментарий к обращению, а также служебные технические поля: URL страницы, referrer, источник обращения и UTM-метки.

3. Цель обработки

Ответ на обращения, обработка B2B-заявок, сопровождение запросов по товару, поддержка, претензии и коммуникация по каналам, указанным пользователем.

4. Контакты по вопросам данных

До публикации полного юридического блока используйте email hello@globalbasket.ru или страницу контактов для запросов, связанных с персональными данными.
